Huffman Codes

Huffman Codes is a particular type of optimal prefix code. It is very useful for file compression. The idea is to assign variable-length codes to input characters, lengths of the assigned codes are based on the frequencies of corresponding characters. The most frequent character gets the smallest code and the least frequent character gets the largest code.

An example of Huffman algorithm
